### 项目简介

本项目是用Python3+Django搭建的一个校园二手市场信息发布平台。前端使用Bootstrap.

### 项目安装

先安装Django框架，具体百度一下.

### 运行
```
python manage.py runserver
```
浏览器输入：http://127.0.0.1:8000/
